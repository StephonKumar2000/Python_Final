Techniques

The techniques I used in this project were utilizing pandas, seaborn, and matplotlib. I used a cv file from an NYC public site to get the data for the resturant inspections. For each graph I used different ways to display the information. For the bar graph I used plt.style.use("fivethirtyeight") to display it. For the pie graph I used matplotlib and displayed it using dropna. The 5 pie graph data was displayed using an axis setup. And for the data in csv file, I filtered the data using some commands to keep specific columns and remove any of unrelevant data from the project.
